# 基础介绍

AI 小说创作工作台是一套帮助你把灵感推进成长篇小说的创作工具。你不需要一开始就会写大纲、做人物表、设计卷结构，系统会把这些步骤拆开，并在每一步给出可以继续推进的结果。

## 你可以用它做什么

- 从一句灵感开始，整理出题材、卖点、读者期待和书名方向。
- 让 AI 自动导演帮你准备世界观、核心角色、卷规划和章节任务。
- 逐章生成正文，并在生成后继续做审核、修复和状态同步。
- 把参考资料、拆书结果和写法样本保存下来，让后续章节可以继续使用。
- 维护角色、世界、题材、推进模式、知识库和模型设置，让整本书更容易保持一致。

## 它适合谁

如果你想写一本小说，但经常卡在“我只有一个点子，不知道后面怎么展开”，这个项目会比较适合你。

它尤其适合：

- 第一次尝试写长篇小说的新手。
- 想用 AI 帮自己搭建故事结构的人。
- 想把参考作品、资料和写法整理成可复用资产的人。
- 想研究 AI 创作流程、自动导演、知识库和长篇生产链的开发者。

## 推荐从哪里开始

第一次使用时，不建议先研究所有设置。更顺的路径是：

1. 打开系统设置，配置一个可用模型。
2. 进入新手上路或小说列表，创建一本测试小说。
3. 输入一句灵感，让自动导演生成方向候选。
4. 选择你更喜欢的方向，继续推进世界、角色和章节准备。
5. 进入章节执行，先跑通第一章。

跑通第一章后，再去看知识库、拆书、写法引擎、题材库和模型路由，会更容易理解这些模块为什么有用。
